FEN strings represent the current position. R = Rook, N = Knight, B = Bishop, Q = Queen, K = King, and P = Pawn.
Lowercase letters indicate a black piece, while uppcase letters indicate a white piece. A slash represents the end of a row, and a number represents that number of consecutive blank spaces.
You can use this to create unique starting positions, and also to create starting positions for games on different sized boards.
If the code you enter is invalid, nothing will happen when you press "update."

Pro tip, you will regret creating boards larger than 200x200.
